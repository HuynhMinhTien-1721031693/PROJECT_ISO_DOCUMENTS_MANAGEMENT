Roadmap hoàn thành dự án ISO Documents Management kèm yêu cầu Android APK

Tiến độ tổng thể ước tính ~30% hoàn thành
Dự án có kiến trúc tốt (Clean Architecture), đã có solution structure và một phần code. Cần hoàn thiện các layer và bổ sung thêm dự án Android.
Đã có: Solution structure, Domain entities, một phần Application layer, WebAPI skeleton, Architecture doc
Chưa có: Infrastructure hoàn chỉnh, Blazor UI, CI/CD, Testing, Android app
Bổ sung mới: Project IsoDoc.MauiApp — .NET MAUI Blazor Hybrid để build APK Android
Công nghệ chính
.NET 8 · ASP.NET Core · MediatR · FluentValidation · EF Core 8 · SQL Server
Azure Blob Storage · Elasticsearch · Redis · Serilog · JWT Bearer
MudBlazor 7.x (Blazor Server UI) — chưa cài đặt
.NET MAUI 8 · MAUI Blazor Hybrid · Android SDK 33+ — dự án mới
01_Domain ~80%
Document, DocumentVersion, ApprovalWorkflow, AuditLog entities
Thêm Domain Events đầy đủ: DocumentApprovedEvent, WorkflowRejectedEvent
Hoàn thiện Value Objects: DocumentCode, Version, Checksum
Thêm Guard clauses trong mọi factory method của Entity
02_Application ~40%
Commands: UploadDocumentCommand, SubmitForApprovalCommand, ApproveDocumentCommand, RejectDocumentCommand
Queries: GetDocumentsQuery (paged), GetDocumentByIdQuery, SearchDocumentsQuery
FluentValidation validators cho tất cả commands
AutoMapper profiles: Document → DocumentDto, Version → VersionDto
Pipeline Behaviors: ValidationBehavior, LoggingBehavior, CachingBehavior
03_Infrastructure ~35%
AppDbContext, Entity configurations (IEntityTypeConfiguration), DB Migrations
Repositories: DocumentRepository, WorkflowRepository, AuditLogRepository
AzureBlobStorageService — upload, download, SAS URL generation
ElasticsearchService — index document, full-text search, suggestions
JwtTokenService — generate, refresh token rotation, lockout
AuditInterceptor EF Core — tự động ghi AuditLogs (immutable)
Email notification service (SMTP / SendGrid) cho workflow events
04_WebAPI ~50%
Program.cs, ExceptionHandlingMiddleware, ApiResponse model
Hoàn thiện DocumentsController (tất cả 8 endpoints + file upload)
WorkflowController, SearchController, AuthController
AuditMiddleware — ghi log IP, UserAgent mỗi request
Swagger/OpenAPI với JWT auth support, phân nhóm endpoint
Health checks: DB, Redis, Elasticsearch, Blob
CORS policy (cho Blazor WASM và MAUI app)
05_Web — Blazor Server ~20%
Cài đặt MudBlazor 7.x, cấu hình theme + dark mode
MainLayout, NavMenu (ẩn/hiện menu theo Role RBAC)
Dashboard.razor — KPI cards, PendingApprovalList, RecentDocuments
DocumentList.razor — MudDataGrid, filter, sort, phân trang
DocumentUpload.razor — wizard upload + chọn workflow
DocumentDetail.razor + DocumentViewer.razor (inline PDF)
ApprovalQueue.razor — danh sách cần duyệt theo Approver
ApprovalDetail.razor — xem tài liệu, ghi comment, approve/reject
Admin pages: UserManagement.razor, RoleAssignment.razor
SignalR realtime notifications (workflow status, new document)
PDF viewer tích hợp (Syncfusion hoặc PSPDFKit)
Services & State Management
IDocumentService, IWorkflowService — HTTP client gọi API
AuthenticationStateProvider tích hợp JWT
Blazored.LocalStorage — lưu UI state, filter preferences
★ Dự án mới: IsoDoc.MauiApp Bổ sung mới
Sử dụng .NET MAUI Blazor Hybrid — tái sử dụng Blazor components, build APK Android natively, kết nối cùng ASP.NET Core Web API backend.
Tạo project: dotnet new maui-blazor -n IsoDoc.MauiApp trong thư mục src/06_MauiApp/
Thêm vào solution: IsoDocumentManagement.sln → Add Project → IsoDoc.MauiApp
Target framework: net8.0-android (tối thiểu Android API 24 / Android 7.0)
Yêu cầu môi trường build APK Bắt buộc
Máy build: Windows 10/11 hoặc macOS 13+ (Ventura)
.NET SDK 8.0.x với workload MAUI: dotnet workload install maui-android
Android SDK: API Level 33+ (Android 13), Build Tools 33.0.x, Platform Tools
Java JDK 11+ (OpenJDK 17 khuyến nghị) — cần để compile Android
Visual Studio 2022 v17.8+ (có Mobile development workload) hoặc VS Code + C# DevKit
Android Emulator hoặc thiết bị thật để test (bật USB Debugging)
NuGet packages cần thêm vào IsoDoc.MauiApp
Microsoft.Maui.Controls 8.x — core MAUI framework
Microsoft.AspNetCore.Components.WebView.Maui 8.x — Blazor Hybrid host
MudBlazor 7.x — tái sử dụng components từ Blazor Server
Microsoft.Extensions.Http — HttpClient để gọi Web API
System.IdentityModel.Tokens.Jwt — lưu và parse JWT token
Blazored.SecureStorage — lưu token an toàn trên thiết bị Android
Microsoft.AspNetCore.SignalR.Client — realtime notifications
Android Manifest permissions cần khai báo

READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE — download tài liệu
CAMERA (tuỳ chọn) — scan QR code tài liệu
POST_NOTIFICATIONS — push notifications workflow (Android 13+)
Cấu trúc thư mục IsoDoc.MauiApp
MauiProgram.cs — đăng ký BlazorWebView, HttpClient, services
Pages/ — tái sử dụng Blazor pages từ IsoDoc.BlazorApp (hoặc shared library)
Services/ApiService.cs — HTTP client gọi https://<api-host>/api/v1/
Services/SecureStorageService.cs — lưu JWT token bằng Android Keystore
Platforms/Android/AndroidManifest.xml — permissions, app metadata
wwwroot/ — Blazor static assets (CSS, JS)
Build & ký APK
Debug APK: dotnet build -f net8.0-android -c Debug
Release APK (cần keystore): dotnet publish -f net8.0-android -c Release
Tạo keystore: keytool -genkey -v -keystore isodms.keystore -alias isodms -keyalg RSA -keysize 2048 -validity 10000
Khai báo trong .csproj: AndroidKeyStore, AndroidSigningKeyAlias, AndroidSigningStorePass
APK output: bin/Release/net8.0-android/publish/*.apk
Testing Chưa bắt đầu
IsoDoc.Domain.Tests — unit test Domain entities, rules, events (xUnit)
IsoDoc.Application.Tests — test handlers, validators, mapping (Moq + FluentAssertions)
IsoDoc.Integration.Tests — test API endpoints (WebApplicationFactory + Testcontainers)
IsoDoc.MauiApp.Tests — unit test MAUI services, navigation logic
Database & Migrations
Tạo initial migration: dotnet ef migrations add InitialSchema
Seed dữ liệu mẫu: Roles (SystemAdmin, ISOManager, Controller, Approver, Viewer)
Full-text index trên Documents (Title, Tags) — SQL Server FULLTEXT CATALOG
Covering indexes theo design spec (IX_Documents_Status_ISO, ...)
CI/CD & Docker
Dockerfile cho IsoDoc.WebAPI (multi-stage build)
docker-compose.yml: API + SQL Server + Redis + Elasticsearch
GitHub Actions workflow: build → test → publish WebAPI
GitHub Actions: build Android APK → artifact upload (mỗi push vào main)
appsettings.template.json → điền đủ ConnectionStrings, BlobStorage, JWT, Elasticsearch
Security (ISO 27001)
Rate limiting middleware (chống brute-force login)
AuditLog immutability — append-only policy, không cho UPDATE/DELETE
SHA-256 checksum verify khi download file (kiểm tra tính toàn vẹn)
HTTPS enforced, HSTS, secure cookie settings
Android: Certificate pinning cho API calls (tránh MITM attacks)
Android: JWT lưu bằng Android Keystore (không dùng SharedPreferences plain text)

Ghi chú quan trọng
MAUI Blazor Hybrid yêu cầu tách Blazor components ra IsoDoc.BlazorShared (Razor Class Library) để dùng chung giữa BlazorServer và MauiApp.
Android app kết nối API qua HTTPS — cần cấu hình base URL trong appsettings.json của MAUI (dev: http://10.0.2.2:5000 cho emulator).
Ưu tiên hoàn thiện Backend API hoàn chỉnh trước khi bắt đầu Android — MAUI app phụ thuộc hoàn toàn vào API.
